# 盲打防线 - 关卡设计方案

## 设计原则

### 核心理念
- **打字为核心**：每关初始金币为0，纯靠防御塔无法通关，必须通过打字击杀怪物获取金币
- **循序渐进**：从基准键到上/下/数字行，再到标点符号，逐步扩大字母范围
- **地图变化**：每5-10关更换地图，增加新鲜感和策略变化
- **难度曲线**：怪物数量、速度、血量逐步提升，防御塔选择逐步解锁

### 金币经济设计
| 怪物类型 | 血量 | 奖励金币 | 击杀所需打字次数 |
|---------|------|---------|----------------|
| 史莱姆 | 30 | 5 | 3次（每次10伤害） |
| 疾行者 | 20 | 8 | 2次 |
| 重甲兵 | 100 | 15 | 10次 |
| Boss | 500 | 100 | 50次 |

**设计思路**：
- 玩家必须打字击杀怪物才能获得金币建造防御塔
- 打字是主要输出，防御塔是辅助和补充
- 每击杀3-4只史莱姆可以建造一座箭塔（50金币）

## 地图设计

### 地图类型一览

| 地图名称 | 路径特点 | 适合关卡 | 战术要点 |
|---------|---------|---------|---------|
| **Z字形** | 两个大转弯，路径中等 | 1-2 | 转弯处放塔效益高 |
| **S形** | 三个转弯，路径较长 | 3-5 | 中间区域塔可覆盖多段 |
| **U形** | 底部大转弯，路径长 | 6-7 | 转弯处放减速塔 |
| **蛇形** | 多次折返，路径最长 | 8-10 | 密集放塔区 |
| **螺旋形** | 向内盘旋，路径长 | 11-12 | 中心塔覆盖范围广 |
| **L形** | 单次大转弯，路径中 | 13-14 | 转角处是关键 |
| **双L形** | 两次转弯，路径较长 | 15-16 | 两个转角都要布防 |
| **锯齿形** | 多次小转弯，路径长 | 17-18 | 塔可以覆盖多段 |
| **迷宫形** | 复杂路径，多次折返 | 19-21 | 需要全局规划 |
| **环形** | 圆形路径，绕一圈 | 22-24 | 内圈外圈都可放塔 |
| **双环形** | 两个环相连，路径极长 | 25-26 | 多重防线 |
| **长线形** | 超长直线+少量转弯 | 27-29 | 需要纵深防御 |
| **综合迷宫** | 超复杂路径 | 30-34 | 全面考验策略 |
| **进阶版地图** | 基础版的复杂版 | 35-44 | 路径更长更复杂 |
| **终极迷宫** | 最复杂路径 | 45-52 | 终极考验 |

## 难度平衡设计

### 打字伤害机制
- 每次正确输入造成 10 点伤害（基础值）
- 连击加成：最高2倍伤害（20点）
- 目标：最前方的怪物

### 经济平衡公式
- **通关所需打字击杀数** ≈ 怪物总数的 60-70%
- **防御塔击杀** ≈ 怪物总数的 30-40%
- **金币获取**：主要靠打字击杀，防御塔击杀奖励相同

### 单关难度参数
- 怪物总数：从18只（第1关）到200+只（第52关）
- 怪物类型占比：史莱姆逐渐减少，疾行者和重甲兵逐渐增加
- 波次数：从3波到25波
- 波次间隔：逐渐缩短

## 无尽模式设计

### 波次循环机制
- 使用全字符范围（字母+数字+标点）
- 每10波为一个循环
- 每个循环怪物血量+20%，数量+10%
- 初始金币：300（无尽模式有启动资金）
- 初始生命：10

### 无尽模式波次结构
| 波次范围 | 怪物特点 | 难度系数 |
|---------|---------|---------|
| 1-10 | 史莱姆为主，少量疾行者 | 1.0x |
| 11-20 | 加入重甲兵 | 1.2x |
| 21-30 | 疾行者增多 | 1.4x |
| 31-40 | 重甲兵占比提升 | 1.6x |
| 41-50 | 每10波出现Boss | 1.8x |
| 51+ | 全类型混合，难度持续提升 | 2.0x+ |

## 关卡设计文档说明

本文档为关卡设计方案，具体实现时需要：

1. **配置文件**：`src/data/levels.json`
2. **怪物配置**：`src/data/monsters.json`
3. **防御塔配置**：`src/data/towers.json`

### 关卡配置字段说明

```json
{
  "id": 1,                    // 关卡ID
  "name": "左手基准键",         // 关卡名称
  "practiceLetters": "asdf",    // 练习字符集合
  "startGold": 0,              // 初始金币（固定为0）
  "startLives": 10,             // 初始生命值
  "typingDifficulty": 1,        // 打字难度（单字母=1）
  "path": [...],               // 怪物路径坐标
  "availableTowers": ["arrow"], // 可用防御塔类型
  "waves": [...]               // 波次配置
}
```

### 波次配置示例

```json
{
  "delay": 0,                   // 波次开始延迟(ms)
  "monsters": [
    {
      "type": "slime",          // 怪物类型
      "count": 5,               // 数量
      "interval": 2500          // 生成间隔(ms)
    }
  ]
}
```

### 注意事项
- `practiceLetters` 字段支持字母、数字、标点符号等所有可打印字符
- Shift相关符号（如大写字母、!@#$等）需要确认游戏引擎是否支持Shift输入
- 如果不支持Shift组合输入，可将标点符号关卡调整为无需Shift的字符

## 调优建议

1. **前12关（字母学习期）**：以教学为主，怪物血量和速度可以稍低，确保新玩家能上手
2. **第13-24关（数字+标点）**：难度平稳提升，每类新字符第一关可以稍简单
3. **第25-44关（速度+精通）**：快速提升期，需要玩家有较好的盲打基础
4. **第45-52关（大师阶段）**：终极挑战，难度可以设置得较高

5. **金币微调**：如果某关太难，可以增加怪物金币奖励；如果太简单，可以减少
6. **怪物数量**：打字练习是核心，宁多勿少，但要注意节奏，给玩家喘息时间
7. **地图变化**：每次换地图后第一关难度可以稍低，让玩家适应新地图
8. **新字符引入**：引入新字符组的第一关，怪物难度可以稍低，让玩家熟悉新字符
